Useful tools for Web Development

The list of items below are an unscientific sample of the tools we love to use when building rich web applications. Try them out. If you have any other favourites that you think we should try out, then let us know at jaxer@aptana.com
  • Aptana IDE The best IDE for Ajax development
  • SQLyog Great win32 DB tools for MySQL
  • Firefox 2 Cross-platform, wide-spread browser that's great for developers
  • Firebug Extension The best Firefox extension for developers
  • IE extension IE preview extension for Firefox
  • Logfile View Logfile Viewer plugin for Aptana IDE or Eclipse (here's a handy colorizer config file)
  • LogWatcher An alternate log watcher plugin for Aptana IDE or Eclipse
